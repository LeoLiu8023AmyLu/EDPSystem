# EDPSystem
EDPSystem 安夏电子席卡 Android 版本 客户端<br>

##外观要求：

	1.轻便可折叠，便携易存储<br>
	2.双显示屏 内侧显示屏为触屏操控 (另外可以有亮度调节功能)<br>
	
##功能要求：

	1.完成席卡作用(设置+显示)<br>
	2.文件共享<br>
	3.网络通信<br>
	4.自动化设置<br>
	5.开机自动启动预定程序<br>
	
##硬性要求：

	1.续航能力<br>
	2.多场景适应能力<br>
