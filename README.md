# EDPSystem
EDPSystem 安夏电子席卡 Android 版本 客户端

## 外观要求：

	1.轻便可折叠，便携易存储
	2.双显示屏 内侧显示屏为触屏操控 (另外可以有亮度调节功能)
	
## 功能要求：

	1.完成席卡作用(设置+显示)
	2.文件共享
	3.网络通信
	4.自动化设置
	5.开机自动启动预定程序
	
## 硬性要求：

	1.续航能力
	2.多场景适应能力
