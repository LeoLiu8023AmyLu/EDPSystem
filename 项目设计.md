# EDPSystem</br>
EDPSystem 安夏电子席卡 Android 版本 硬件客户端 (题词机项目) </br>

## 硬件外观要求：</br>

1. 轻便可折叠，便携易摆放</br>
2. 内侧显示屏为触屏操控 (另外可以有亮度调节功能)</br>	
3. 内置充电电池</br>
4. 外部有两个**USB接口**,**网线接口**,**电源充电接口** </br>

## 功能要求：</br>

1. 完成席卡作用(设置+显示)</br>
2. U盘文件读取</br>
3. 网络通信</br>
4. 自动化设置</br>
5. 开机自动启动预定程序</br>

## 硬性要求：</br>
1. 续航能力 8H 以上</br>
2. 多场景适应能力 ** 不同会议室都可以使用 ** </br>
